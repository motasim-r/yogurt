# Yogurt Desktop App

Electron + React desktop app for the unified Yogurt monorepo.

This app is now the canonical UI/control center for:

1. Granola MCP auth/sync/extraction
2. Task persistence and feed rendering
3. IronClaw task execution and realtime run updates

## Local commands (from this folder)

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run test`
- `npm run typecheck`

For root-level workflow, use commands from:
`/Users/motasimrahman/Desktop/codex/yogurt`.
